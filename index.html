Oxidized Git

Features

  • Runs on Windows, Mac, and Linux
  • Git Operations
  • Commit Graph to see your history at a glance!
  • Create/Delete/Checkout Branches
  • Commit
  • Fetch
  • Pull (automatically detects and performs either a fast-forward merge or a rebase!)
  • Push (with option to Force Push)
  • Save and Apply Stashes
  • View File Diffs in a Commit
  • Stage/Unstage and View Changes
  • Discard Changes
  • Merge
  • Rebase
  • Cherrypick
  • Revert
  • Reset (Soft, Mixed, or Hard)

Features Planned For A Future Release

  • Interactive Rebase
  • Better Conflict Resolution
  • Open Multiple Repos Simultaneously
  • Stage/Unstage/Discard Individual Lines and/or Hunks
  • Sort Graph by Date
  • Search Commit Graph
