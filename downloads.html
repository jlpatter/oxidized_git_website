Notes about the Windows version

The Windows version is currently unsiged (code signing is in progress) and will appear as if it is from an unknown publisher. You may see the "Windows SmartScreen" window appear. This is because Windows needs to "build trust" with the application. The more users that use this, the less often it will appear. So tell your friends to start using Oxidized Git today!
